Then I decided to learn to program with Python. I saw all the possibilities that you have with this programming language and I was definitely excited about what was further ahead. I spent time teaching myself to develop Web Applications with Django, JavaScript, jQuery and of course HTML and CSS. Then I went on to learn how to Web Scrape with the Python modules - BeautifulSoup and Selenium, because of an idea that I had for a website.

And so... here I am now, still excited for the future, waiting for more problems to solve! :)
